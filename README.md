# Personal Portfolio Website

Live Site: https://ajene-dev.vercel.app/

My personal portfolio website built with React, TypeScript, Tailwind CSS, and Vite.

The goal of this project is to serve as a digital business card that showcases my projects, skills, background, and contact information while providing a polished user experience.

## Features

- Responsive single-page design
- Modern glassmorphism UI
- Animated background glow effects
- Featured project showcase
- About Me section
- Contact form powered by Formspree
- Smooth scrolling navigation
- Mobile-friendly layout

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

### Deployment

- Vercel

### Forms

- [Formspree](https://formspree.io/)

## Local Development

Clone the repository:

```bash
git clone <repo-url>
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
